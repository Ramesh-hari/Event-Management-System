##

A modern Event Management System built to simplify organizing, managing, and attending events.
The platform enables admins to manage events & speakers, while users can browse and register for events seamlessly.

##

## ✨ Key Features
### For Users
- User Registration & Login – JWT-secured authentication
- Event Browsing – View event details (date, venue, speakers, description)
- Search & Filters – Find events by category, location, or date
- Event Registration – Register for events instantly
- Responsive UI – Smooth experience on all devices

### For Admins
- Admin Dashboard – Manage full event lifecycle
- Event Management – Create, update, and delete events
- Speaker Management – Add or update speaker details
- Attendance Tracking – View registered users
- Role-based Access – Admin-only capabilities

## Tech Stack
- Frontend: React 19, Bootstrap CSS, React Hook Form, Framer Motion (animations)
- Backend: Spring Boot 3.5.6, REST API routes, MySQL database
- Security: JWT authentication, bcrypt password hashing

## Deployment
- Frontend: Netlify
- Backend: Railway
- Database: MySQL (Railway)
